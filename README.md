# 🚀 PROJECT-Q-30-Days-Quantum-Challenge

A structured 30-Day Quantum Computing Challenge where I document my daily learning journey, practical implementations, and mini-projects using **Qiskit**, **IBM Quantum**, and Python.

The objective of this challenge is to build a strong foundation in quantum computing by combining theory with hands-on coding projects.

# 🎯 Goals

- Learn Quantum Computing from fundamentals
- Understand qubits, superposition and entanglement
- Build quantum circuits using Qiskit
- Work with quantum gates
- Perform quantum measurements
- Develop mini projects
- Document the complete learning journey publicly
- Build a portfolio of quantum computing projects

## ✅ Day 07 — Quantum Randomness Laboratory

Built a Quantum Random Number Generator (QRNG) using Qiskit.

Features:

- Generates quantum random bits
- Executes multiple measurement shots
- Displays probability distributions
- Visualizes measurement results using histograms
- Compares quantum randomness with Python's classical random generator
- Calculates measurement statistics
